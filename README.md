# 📘 Introdução à Linguagem C
### 📌 Descrição

Programa simples em C que solicita ao usuário:

    Idade

    Nome

    Altura

## Uma opção (caractere)

E depois exibe as informações digitadas na tela.

Projeto desenvolvido para praticar:

Entrada de dados com scanf

Saída de dados com printf

Uso de variáveis (int, float, char)

Estrutura básica do main
